# Описание базы данных образовательной платформы

В рамках проекта реализована схема данных и таблицы для образовательной платформы.

## Краткое описание структуры БД

Созданы следующие таблицы:

- **Пользователи учебной платформы** - информация о студентах, преподавателях и администраторах
- **Курсы** - основные данные о курсах (название, описание, преподаватель, статус)
- **Модули курсов** - разделы внутри курса
- **Уроки курсов** - отдельные учебные единицы внутри модулей
- **Записи на курсы** - информация о регистрации студентов на курсы
- **Прогресс прохождения** - отслеживание выполнения уроков студентами
- **Тесты** - контрольные задания и экзамены
- **Вопросы по тестам** - структура тестовых заданий
- **Ответы по тестам** - варианты ответов и правильные решения
- **Сведения о выданных сертификатах** - документы об окончании курсов

## Возможности системы

С помощью данной структуры можно:

1. **Точно отслеживать прогресс студентов**
   - По отдельным сегментам курса (модулям, урокам)
   - По общему прохождению курса
   - По результатам тестирования

2. **Анализировать эффективность обучения**
   - Успеваемость студентов
   - Завершаемость курсов
   - Качество учебных материалов

3. **Управлять образовательным процессом**
   - Выдавать сертификаты
   - Контролировать выполнение заданий
   - Оценивать знания студентов
